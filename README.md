## Merchant Ops Dashboard

# How to run
1. npm install
2. npm run dev
3. Open `http://localhost:5173`

# Tech used
- React (Vite)
- React Router
- Tailwind 
- Recharts

# Features
- Dashboard with summary stats + chart
- Merchants table/listing with search, filter, sort
- Merchant detail modal with update rules & confirmation
- Add/Edit merchant with validation
- Responsive design
